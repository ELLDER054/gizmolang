Building Gizmo From Source

Requirements:

    

  1. A terminal or shell (An Interface For Users To Interact With The Basic Parts of a Computer Core)
  2. clang (The C Compiler)
  3. llvm (llc)
  4. cmake (The Program To Build or Make C Programs)
  5. git (The Tool To Help You Organize Different Versions of Your Program)
  6. gcc (Another C Compiler)
    7. 



Run the following four commands in your shell or terminal to build Gizmo:

1. git clone https://github.com/ellder054/gizmo-lang.git
2. cd gizmo-lang
3. mkdir build
4. cd build
3. cmake ..
4. make
