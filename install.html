The Gizmo Installation Guide

Linux is recommended for Gizmo users, but if you are on
windows, Ubuntu is just as good.

If git is already installed, you can skip the first
command

sudo apt install git
git clone https://github.com/ellder054/gizmo-lang
cd gizmo-lang
./main.sh filename.gizmo output
./output
