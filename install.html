The Gizmo Installation Guide

1. If you don't have a Linux machine, go see the Ubuntu Installation Guide.

2. Once you have a terminal up, you can proceed to install git. (If you already have git, you can skip this step)
	
	sudo apt-get install git-all
	
	
3. Next, copy the following commands to build Gizmo.
	
	git clone https://github.com/ellder054/gizmo-lang.git
	mkdir gizmo-build
	cd gizmo-build
	cmake ../gizmo-lang
	make
	

4. (Optional) Compiling and executing a Gizmo program is easy too!

 - Navigate to the gizmo-build directory
 - run "./gizmo filename.gizmo out", but replace "filename" with your input file and "out" with your output file.
 - run "./out" (replace out with your output file again) to execute your file!
