Building Gizmo From Source

Requirements:

    

  1. A terminal or shell
  2. clang
  3. llvm (llc)
  4. cmake
  5. git
  6. gcc
    7. 



Run the following commands to build Gizmo:

    git clone https://github.com/ellder054/gizmo-lang.git
    mkdir gizmo-build
    cd gizmo-build
    cmake ../gizmo-lang
    make
