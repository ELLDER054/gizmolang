The Gizmo Packaging Guide

WARNING: Please Be Aware that this is not a finished website, and these commands will NOT yet work!

The Gizmo Packaging System will allow any Gizmo repository to be a package if the name of the repository starts
with "gizmo-" and ends with "-package"
To add a package, follow these commands:
 - If you don't have a repository with the rules above, make one
 - In your terminal, type:
  
  ./gizmops link project
  
  replace "link" with the link to your github project and replace "project" with your Gizmo file
