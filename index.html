The Gizmo Programming Language Docs Install Repository 
Gizmo is a systems programming language with many modern features.
Some features of Gizmo are:

1. classes and inheritance
2. case statements
3. statically typed variables and functions

Gizmoc, Gizmo's compiler, has some features too.

1. C front-end
2. LLVM backend
3. Smart parser which combines phase 2 of a compiler with phase 3 of the compiler
4. Small scanner
5. Small standard library

Gizmo is available in your browser!
