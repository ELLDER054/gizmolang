Output

As always, the first step in learning a programming language is printing a desired output to the screen.

Gizmo has a feature that allows you to do this

write("Hello, Gizmo!");

Entering this code into your Gizmo editor will print:

Hello, Gizmo!

Using this method, you can also print any data type (See data types section for information on data types).

For example, we can print either a number, a decimal number, or a character onto the screen

write(5); // A number, or an integer

write(10.67); // A decimal number

write('a'); // A character
