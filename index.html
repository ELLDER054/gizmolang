The Gizmo Programming Language

What is Gizmo

Gizmo is a small programming language that is easy to use and fast.
It can also be portable if you use the handy virtual machine.
Gizmo also has many helpful features like switch-case statements, strings, classes (plus inheritance), and much more!

Why Did You Make Gizmo and Why should I use it?

I admit to being a little blind about the knowledge of how programming languages worked. This project is how I got a little experience
on the "making" side of programming languages as oppose to the "using" side.
