The Gizmo Programming Language Docs Install Repository 
Gizmo is a systems programming language with many modern features.
With Gizmo, many projects that involve parsing strings on a low level become
much easier.
A quick peek:


\ fizzbuzz written in Gizmo
int i = 0;
while (i < 100) {
	if i % 2 == 0 and i % 3 == 0 {
		write("fizzbuzz");
	else if i % 2 == 0 {
		write("fizz");
	else if i % 3 == 0 {
		write("buzz");
	}
	i++;
}


From this code, some key parts of Gizmo's syntax and coding convention are revealed.
Comments begin with the '\' character and continue to the end of the line.
Names should be snake_case.
"While" loops are generally preffered to "for" loops, unless you are making a "for in" loop.
Most of the time, you have to specify which type a variable or
function returns. See this page
for more info about this.
