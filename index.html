The Gizmo Programming Language

What is Gizmo

Gizmo is a small programming language that is easy to use yet portable and fast. Gizmo also has most helpful features like switch-case statements, strings, classes (and inheritance), and much more!

What makes Gizmo different from C?

Some of the benefits of using Gizmo over C or some other languages:
  1. You don't have to worry about memory management and pointers
  2. It is fast
  3. It is portable (anywhere C can be run, Gizmo can be run)
  4. Lots of features
