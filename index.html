The Gizmo Programming Language Docs Install Repository 
Gizmo is a small, user-friendly programming language with a focus on low level
string manipulation.


Contributions

Contributions are welcomed in the development of Gizmo. Some ways to help are:
1. Use Gizmo in your applications
2. Make modules and libraries for other Gizmo programmers
3. Contribute to the actual repository (click the "Repository" tab)
4. Sponser us on github


Why should I use Gizmo

Gizmo is a lot like C, but it does all memory allocations for you as opposed to
having to do it all yourself.
This makes Gizmo a goto language when it comes to fast and low level
string manipulation.
