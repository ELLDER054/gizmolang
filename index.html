The Gizmo Programming Language

What is Gizmo?

Gizmo is a small programming language that is easy to use and fast.
It can also be portable if you use the handy virtual machine.
Gizmo also has many helpful features like switch-case statements,
strings, classes,
and much more!

Why should I use it?

Gizmo can help you if you need string manipulation on a low level.
A few examples that would benefit from this are:
  1. Compilers
  2. Fuzzers
  3. Operating Systems
