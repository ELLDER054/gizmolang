The Gizmo Documentation

Gizmo is a simple language, but we still don't expect you to know everything about
Gizmo right off the bat. This entire page is dedicated to teaching
Gizmo to newcomers and seasoned programmers alike. If you have a question or you don't
understand something, ask a question on Stack Overflow with the tag "gizmo-lang".


Hello, Gizmo!

Like most languages, the classic hello world program is a great way to jump into Gizmo.
Add the following code into your coding environment.


1)	write("Hello, Gizmo!");

	Run your project and you should see "Hello, World!" pop up in the console.
Congrats, you have just created your first Gizmo program!

Let's make our program a little more complicated.


1)	string input = read("Enter some input: ");
2)	write("You entered '" + input + "'");



Variables and Input

	A variable is a container. The value it contains can be whatever
you assign it to.
In the previous example, we 'made a container' and made it contain
whatever 'read("Enter some input: ")' is.

'read' is a function that reads input from the console, with the the value
you give it as a prompt.
Thus, line 1 of the code sets the value of "input" to whatever the user
enters as input.
	
	Line 2 of the code is similar to our first code. We are printing
the words "You entered '", then we are
printing the value of variable "input", then we are printing "'".

	Go ahead and run your code. It should be waiting for you to
answer "Enter some input: "
If you enter "test" and click enter, it should print "You entered 'test'".
