The Gizmo Programming Language Docs Install Repository 
Hello, Gizmo!

Like with most languages, the classic "Hello, World!" program is a great starting point.
Let's print out "Hello, Gizmo" to learn more about Gizmo.

    hello.gizmo

write("Hello, Gizmo!");


Compile and run your file.

./gizmo hello.gizmo out
./out
Input
The read function is used to read input from the user. It's first argument
is a prompt. Try it out by asking the user for a fruit and printing it out.

    fruits.gizmo

write(read("Enter a fruit: "));

Again, compile and run your project. This time, your shell should be
waiting for you to answer the question.
If you type something in and hit enter, it will print whatever you typed in.
